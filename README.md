# Contact Center AI Analytics

AI powered analytics platform for contact centers.

This project simulates a modern AI speech analytics system used to analyze customer service calls.

## Features

- Call transcript analysis
- Sentiment detection
- Intent recognition
- Agent performance monitoring
- Quality scorecards
- Operational dashboards

## Architecture

Telephony → Data Pipeline → AI Analysis → Data Warehouse → BI Dashboards → Management App

## Modules

Executive Dashboard  
Agent Performance  
Call Explorer  
AI Insights  
Quality Monitoring  

## Technologies

- Base44 (low-code application platform)
- relational data model
- AI sentiment analysis
- BI dashboards

## Demo

This project simulates:

- 10 teams
- 80 agents
- 2000 customers
- 10000 calls
- AI conversation analytics
